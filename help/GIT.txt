GIT:

1) cd Директоря (Выбор папки с проектом)
2) ls (список файлов)
3) git init (создаёт папку .git, для того чтобы распознавать проект) - обязательно
4) cd .. (сбрасывает директорию на 1 ветку вниз)
5) git status (Просмотр статуса проекта)
6) git add . (Добавляет все файлы в папке в проект)
Если нужен определённый файл git add название файла
7) git commit -m "название" (создаёт типо сейв)
8) можно создать файл в папке с названием .gitignore (записываю туда названия файлов, они буду игнорироваться git)
9) git branch (просмотр веток)
10) git branch Название (создаёт ветку)
11) git branch -D Название (удаляет ветку)
12) git checkout Название (переключение на другую ветку)
13) git checkout -b Название (Создаёт ветку и сразу на неё переключается)
14) если создать файл и закоммитить его на другой ветви, то переключившись на другую ветку, этого файла там не будет
15) git merge Название ветки (совмещает ветку с той, на которой ты находишься)
16) Имя пользователя (git config —global user.name)
17) Почта пользователя (git config —global user.email)
18) git remote add origin Ссылка с GitHub
git push -u origin master
(перекидывает проект на сервера GitHub)
19) mkdir Название (Создаёт папку)
20) git clone Ссылка с GitHub (Создаёт клон проекта с GitHub)
21) git pull Выгрузить изменения файлов c GitHub
