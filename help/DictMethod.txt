Методы словарей:
	
	1) Метод .clear() очищает весь словарь. В итоге после вызова получится пустой словарь

	2) Метод get(key, text) – позволяет получить значение ключа.  Нужно указать внутри скобок один аргумент – ключ, значение которого хотим получить. Если ключа в словаре нет, то выведет None, но если в метод get() внести второй аргумент, то вместо None будет появляться это значение

	3) Метод setdefault()  - получает значение ключа. Похож на прошлый метод get(), однако при обращении к несуществующему ключу он вносит в словарь новую пару ключ-значение. Значением будет второй аргумент, который был передан в этот метод, либо же None, если в методе только один аргумент

	4) Метод pop() – возвращает значение, находящееся под указанным ключом, а из самого словаря удаляется пара с данным ключом

	5) Метод popitem() удалит и вернет двойной кортеж (key, value) из словаря. Пары возвращаются с конца словаря, в порядке LIFO (последним пришёл - первым ушёл).

	6) Метод keys() позволяет получить все ключи словаря.

	7) Метод values() позволяет получить все значения словаря

	8) Метод items() – возвращает коллекцию, в которой содержатся все пары «ключ-значение» в виде кортежей

	9) Метод update() обновляет словарь элементами из другого словаря. Другими словами, метод сливает(мержит от английского «merge») один словарь в другой: добавляются новые ключи из другого словаря, при совпадении ключей записывается значение из переданного словаря
